# Maker Faire Tokyo 2017<br>FXAT勉強会チーム

3D LEDのSDKとコンテンツを格納するリポジトリです。

* lib  
SDKのソースコード  
windows, Mac OS, Linuxに対応  
言語はC++11
* content  
コンテンツのソースコード  
言語はC++, C#, GO, Ruby, Python2, Python3
